# **Informe de Factibilidad**

## 1. Descripción del Proyecto

### 1.1 Nombre del proyecto
Vortex-Normalize: Validador Inteligente de Esquemas Relacionales.

### 1.2 Duración del proyecto
16 semanas.

### 1.3 Descripción
El proyecto consiste en el desarrollo de una herramienta inteligente que permite validar y optimizar esquemas relacionales hasta alcanzar la Forma Normal de Boyce-Codd (BCNF). Utiliza un motor lógico en Python, visualización mediante grafos y soporte de inteligencia artificial para explicar errores de forma comprensible.

### 1.4 Objetivos

#### 1.4.1 Objetivo general
Desarrollar una plataforma web intuitiva que permita validar y optimizar esquemas relacionales.

#### 1.4.2 Objetivos específicos
- Implementar el motor lógico de normalización.
- Visualizar dependencias mediante grafos.
- Integrar IA para explicaciones.
- Almacenar historial de análisis.
- Desplegar el sistema en la web.

---

## 2. Riesgos

- Errores en respuestas de IA.
- Curva de aprendizaje de React Flow.

## 3. Análisis de la Situación actual

### 3.1 Problema
La normalización se realiza manualmente, generando errores, redundancia y bajo rendimiento.

### 3.2 Hardware y software
- Hardware: laptops con 8GB RAM.
- Software: FastAPI, React, MySQL, Docker, APIs IA.
